## рџ“– What is StegoVault?

**StegoVault** is an Android microproject that implements **Image Steganography** вЂ” the technique of hiding secret text messages *inside* digital images without any visible change to the image. The image looks completely normal to the naked eye, but carries a hidden message embedded at the pixel level.

> *"The best place to hide something is in plain sight."*

Unlike encryption (which scrambles data and makes it obviously hidden), steganography makes the very existence of the secret message invisible. StegoVault combines both concepts вЂ” your message is hidden AND undetectable.

## рџ§  How It Works вЂ” The LSB Algorithm

StegoVault uses the **LSB (Least Significant Bit)** steganography technique.

Every pixel in a digital image has three color channels: **Red**, **Green**, and **Blue** вЂ” each stored as an 8-bit number (0вЂ“255). The LSB algorithm replaces the **last bit** of each channel with one bit of the secret message.

```
Original pixel channel:   1 0 1 1 0 1 1 0  (182)
Message bit to hide:                      1
Modified pixel channel:   1 0 1 1 0 1 1 1  (183)  в†ђ change of just 1/255
```

Since the change in value is at most **В±1 out of 255**, the human eye cannot distinguish the modified image from the original. With 3 bits per pixel (one per channel), a 1-megapixel image can hide approximately **375,000 characters** of text.

### Capacity Formula

```
Max characters = (image_width Г— image_height Г— 3) / 8
```

### Decoding Flow


## вљ пёЏ Limitations

- **PNG only** вЂ” JPEG/WebP compression destroys hidden data
- **No password protection** вЂ” anyone with this app can decode a stego image
- **Text only** вЂ” this version supports text messages only (not files or images)
- **Single-layer** вЂ” one message per image; re-encoding overwrites previous data
